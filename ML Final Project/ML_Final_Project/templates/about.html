Generative Adversarial Networks on Crypto Punks

Machine Learning Project Review

As part of computer science studies in grades 11,12 we have been asked to write a machine learning project on a subject of our choice.

In my project i've decided to include personal things I find out interesting such as NFTs.
I have taken 10,000 Pictures of the Crypto Punks NFT Collection and built a special kind of deep neural network that doesn't output a binary output such as True\False but outputs a picture.
My goal was to create a network that will create pictures that would look like they originally belong to the original collection.

The Crypto Punks collection includes 10,000 unique pictures of 24x24 pixels each.
